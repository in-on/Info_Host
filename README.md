# 📡 Info_Host Tool

A simple Python network utility tool that retrieves system and DNS information using the `socket` library.

---

## 🚀 Features

- Display local machine **hostname**
- Show local **IPv4 address**
- Perform **DNS lookup** for a domain
- Perform **reverse DNS lookup** for an IP address
- Simple command-line interface

---

## ⚙️ Requirements

- Python 3.x

No external libraries required (uses built-in modules only).

---

## 📌 Usage

Run the script using:

```bash
python info_host.py <option> <value>


| Option | Description                                       |
| ------ | ------------------------------------------------- |
| 1      | Show system hostname and local IP                 |
| 2      | Get information about a domain (e.g., google.com) |
| 3      | Reverse lookup IP address                         |
| 0      | Exit program                                      |
